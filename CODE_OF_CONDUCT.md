# Code of Conduct

## Our pledge

We want participation in this project to be a friendly, safe, and welcoming
experience for everyone, regardless of background or identity. Everyone who
interacts with this project — in issues, pull requests, discussions, and any
other channel — is expected to act respectfully and in good faith.

## Our standard

This project **adopts the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1** as its code of conduct. The full, canonical text (including the
detailed standards of behavior and the enforcement guidelines) is published here:

- Human-readable: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Plain text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.txt>

In short: be kind and constructive, assume good intent, welcome newcomers, accept
feedback gracefully, and focus on what's best for the community and for the kids
this project is built to help. Behavior that harasses, demeans, or excludes others
is not tolerated.

## Scope

This Code of Conduct applies within all project spaces (the repository, issues,
pull requests, and discussions) and when an individual is officially representing
the project in public spaces.

## Reporting

If you experience or witness unacceptable behavior, report it **privately** — do
**not** open a public issue. Use either:

- GitHub's **private vulnerability reporting** for this repository:
  `https://github.com/jayaregalinada/enilex-math-4-monorepo/security/advisories/new`
  (this is a private channel visible only to maintainers), or
- a direct private message to the maintainer **[@jayaregalinada](https://github.com/jayaregalinada)**.

All reports will be reviewed and investigated promptly and fairly. The maintainer
is obligated to respect the privacy and security of the reporter of any incident.

## Enforcement

The maintainer is responsible for clarifying and enforcing this Code of Conduct
and may take any action they deem appropriate, up to and including a temporary or
permanent ban from the project, for behavior they consider inappropriate,
threatening, or harmful. Enforcement follows the Contributor Covenant v2.1
enforcement guidelines linked above.
