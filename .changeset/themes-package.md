Add the `themes` package: four cosmetic themes (Cosmic, Ocean, Jungle, Candy)
as colour palettes + emoji life icons and mascots, a `ThemeProvider` that
injects the palette as `--color-*` CSS variables, a `useTheme` hook, a
`pickRandomTheme` selector, and a `Mascot` component that reacts to the run
(idle / cheer / sad) via CSS animation. All visuals are colour + emoji — no
image files.
